# Conservación de Primates usando AppInventor

Análisis de Población de Primates con IA y Regresión Lineal: Aplicación móvil desarrollada en MIT App Inventor diseñada para el monitoreo y predicción de poblaciones de primates.

Características principales:
    1. Sincronización en tiempo real: Conexión con Google Sheets para la gestión de datos censales.
    2. Limpieza de Datos: Algoritmos para detección de anomalías y eliminación de entradas erróneas.
    3. Análisis Estadístico: Generación de líneas de tendencia (LOBF), cálculo de coeficientes de correlación y regresión lineal.
    4. IA Generativa: Integración con un modelo de lenguaje (ChatBot) para la interpretación de tendencias y proyecciones de conservación.
